# Swift

## Introduction
A client application of [Amnoon](http://www.amnoon.com) billing system in [SISE](http://www.sise.com.cn/).

See the [details of protocol](https://github.com/xingrz/swiftz-protocal).

### Development Environment
* OpenJdk 9.0.1
* Ubuntu 17.10

### Runtime Environment
* Jre 9 or more
